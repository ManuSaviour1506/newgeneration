# 🎓 New Generation School Website (CMS)
Developed and deployed a full-stack CMS-based web platform for an educational institution using the MERN stack (MongoDB, Express, React, Node.js). 
The system enables non-technical administrators to dynamically manage website content, including news, media, staff, and student information through a dedicated admin dashboard.

Architected a scalable and modular solution with reusable frontend components and structured RESTful APIs.
Implemented efficient media handling workflows and ensured responsive, user-friendly design for public access.
Deployed the application using CDN-based frontend hosting and cloud backend services, optimizing performance, availability, and cost.
A production-ready **full-stack MERN application** built for managing and displaying dynamic school content with a custom admin dashboard.

🌐 Live: https://www.newgenerationeluru.com  

## 🚀 Overview

This project is a **custom Content Management System (CMS)** that allows school administrators to manage website content such as news, staff, students, and media without technical knowledge.

---
## ✨ Features

### 🔐 Admin Panel
- Manage news, events, and announcements
- Upload and manage image gallery
- Add/Edit/Delete staff and student data
- Real-time content updates

### 🌍 Public Website
- Responsive UI (mobile-first)
- Dynamic content rendering from database
- Clean and accessible design
